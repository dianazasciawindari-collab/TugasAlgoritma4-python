✨ 1. Algoritma FizzBuzz ✨

Algoritma ini menggunakan perulangan dan percabangan untuk menampilkan angka dari 1 sampai 100 dengan kondisi tertentu.

💻 Code Python:

for i in range(1, 101):
    if i % 3 == 0 and i % 5 == 0:
        print("FizzBuzz")
    elif i % 3 == 0:
        print("Fizz")
    elif i % 5 == 0:
        print("Buzz")
    else:
        print(i)
    







    
🌷 2. Algoritma Konversi Hari 🌷


    

    Algoritma ini digunakan untuk mengubah jumlah hari menjadi tahun, bulan, 
    dan hari menggunakan operasi pembagian dan sisa bagi.
    


    
💻 Code Python:

    
total_hari = int(input("Masukkan jumlah hari: "))

tahun = total_hari // 365
sisa_hari = total_hari % 365

bulan = sisa_hari // 30
hari = sisa_hari % 30

print("Tahun :", tahun)
print("Bulan :", bulan)
print("Hari  :", hari)
